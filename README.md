# Family Health Chatbox

Repository này đang chứa bộ tài liệu quản lý dự án cho sản phẩm:
- khách hàng: bệnh viện, viện dưỡng lão
- người dùng cuối: bệnh nhân/người cao tuổi, gia đình, y tá, bác sĩ
- mục tiêu: ra MVP nhanh theo Agile để pilot sớm

## Cấu trúc tài liệu

- [PRD.md](/Users/tranbadat/Documents/study-projects/agile/docs/PRD.md)
  Tài liệu sản phẩm tổng thể: bối cảnh, mục tiêu, personas, module, feature, scope MVP

- [BACKLOG.md](/Users/tranbadat/Documents/study-projects/agile/docs/BACKLOG.md)
  Product backlog chi tiết theo `Module -> Feature -> User Story`, có ưu tiên, sprint đề xuất, SP draft

- [SPRINT-PLAN.md](/Users/tranbadat/Documents/study-projects/agile/docs/SPRINT-PLAN.md)
  Kế hoạch triển khai theo phase, sprint, phân công 5 người, KPI và release plan

- [DESIGN-UX.md](/Users/tranbadat/Documents/study-projects/agile/docs/DESIGN-UX.md)
  Tài liệu design thinking, UI/UX cho người cao tuổi, luồng thao tác chính, voice/SOS/hardware direction

## Cách dùng

- nếu cần overview cho stakeholder: mở `PRD.md`
- nếu BA/PO refinement với team: mở `BACKLOG.md`
- nếu planning sprint và phân việc: mở `SPRINT-PLAN.md`
- nếu workshop với designer/product/dev về trải nghiệm: mở `DESIGN-UX.md`
- nếu import sang Notion: copy từng bảng hoặc từng section theo file tương ứng

## Gợi ý import Notion

Nên tách thành các page/database:
- Product Overview
- Personas & Problem
- Module / Feature Map
- MVP Scope
- Product Backlog
- Sprint Plan
- Team Assignment
- Risks & KPI
